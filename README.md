# ucan-man 

VS code extension for inspecting UCAN requests 

# Additional features 

Visual graph for proof chains (tree view)

Highlight expired UCANs

Warn if issuer !== "with" DID

Color-coded capabilities (e.g., store/add, blob/read)

There are two types of UCANs in the real world:

JWT UCAN → what developers paste / see in logs

Invocation UCAN decoding → IPLD objects stored in CAR (UCANTO internal)
